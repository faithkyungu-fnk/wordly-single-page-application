# wordly-single-page-application
wordly-(SPA)



//DESCRIPTION
Wordly Dictionary is a web application that allows users to search for English words and see their meanings, pronunciation, synonyms, and examples. It uses the Free Dictionary API to get word information.


//FEATURES
- Search for any English word
- View the word meaning
- View pronunciation
- Listen to audio pronunciation
- View synonyms
- Save favorite words



//TECHNOLOGY USED
- HTML
- CSS
- JavaScript
- Dictionary API

//HOW TO USE
1. Enter a word in the search box.
2. Click the **Search** button.
3. View the word details.
4. Save the word to your favorites if you like.
